# AWS Migration Hub

## O que é
Painel centralizado para acompanhar progresso de migrações em diferentes ferramentas.

## Quando usar
- Governança de programas de migração com vários times.
- Visão executiva de ondas e status.

## Pontos de prova
- Consolida acompanhamento; não executa todas as migrações sozinho.
- Ajuda em rastreabilidade e priorização.
