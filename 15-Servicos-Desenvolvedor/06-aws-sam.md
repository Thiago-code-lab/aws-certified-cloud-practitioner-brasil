# AWS SAM

## O que é
Framework para desenvolvimento e deploy de aplicações serverless usando templates simplificados sobre CloudFormation.

## Quando usar
- Projetos com Lambda, API Gateway e eventos.
- Fluxo local de desenvolvimento serverless.

## Pontos de prova
- Acelera criação de aplicações serverless padronizadas.
- Integra com pipeline de CI/CD e IaC.

## CLI útil
```bash
aws serverlessrepo list-applications
```
